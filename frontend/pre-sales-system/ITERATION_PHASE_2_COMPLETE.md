## 🚀 部署说明

### 后端部署
```bash
cd backend
npm install
npx prisma generate
npx prisma db push
npm run start:prod
```

### 前端部署
```bash
cd frontend
npm install
npm run build
# 部署dist目录到服务器
```

### 环境变量配置
```env
# 后端
DATABASE_URL=postgresql://...
JWT_SECRET=...
JWT_EXPIRATION=7d

# 前端
VITE_API_BASE_URL=http://...
```
