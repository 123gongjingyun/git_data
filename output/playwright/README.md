# Playwright Artifacts

本目录只保留当前仍有追踪价值的浏览器自动化脚本与最近一轮验收产物。

## 当前保留口径

- `local3-*`
  - 2026-04-11 本地滚动条自动化验收最新基线。
  - 已覆盖工作台、项目管理、商机管理、解决方案、投标管理、合同管理、知识库，以及系统设置中的团队管理、知识库目录管理、菜单权限管理、操作权限管理、飞书集成卡片预览。
- `cloud2-*`
  - 已被 `cloud3-*` 取代，不再保留。
- `cloud3-*`
  - 2026-04-11 云端滚动条自动化验收最新基线。
  - 已与 `local3-*` 对齐覆盖“菜单权限管理 / 操作权限管理 / 飞书集成卡片预览”扩展目标，并确认所有目标页均为 `theme=dark`。
- `01-*` ~ `04-*`
  - 商机审批回归脚本生成的关键过程快照。
- `opportunity-regression-check.mjs`
  - 商机审批回归脚本。
- `opportunity-regression-report.json`
  - 最近一次商机审批回归摘要。
- `scrollbar-theme-check.mjs`
  - 滚动条主题验收脚本。
- `playwright-runtime.mjs`
  - Playwright 运行时解析层，避免脚本绑定临时目录。

## 清理规则

- 被最新一轮同环境、同类型结果完全替代的 `local*` / `cloud*` 历史产物应及时删除。
- MCP 临时快照、人工兜底摘要等只在排障期间临时保留；一旦同轮自动化结果产出，应优先删除。
- 若后续补跑新的云端滚动条扩展验收，建议继续按 `cloud4-*`、`cloud5-*` 递增命名，并在确认通过后删除上一轮同类型云端产物。
